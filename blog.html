Question-1: What are the different ways to select an element in the DOM?

  • getElementById(): Selects one element by its id.
  • getElementsByClassName(): Gets all elements with a specified class name.
  • getElementsByTagName(): Returns all elements with a given tag name.
  • querySelector(): Selects the first element that matches a CSS selector.
  • querySelectorAll(): Returns a list of all the elements those matches a CSS selector.

    • Question-2: What are the differences between innerHTML, innerText, and textContent?

  • innerHTML: The HTML inside of an element, including tags.
  • innerText: The text content of an element, with respect to rendered styles.
  • textContent: The text content of an element and its descendants, returned as a string.

    • Question-3: What is event delegation in DOM?

    Event delegation is adding an event listener to the parent, that fires for the children. This gives better performance, and easy to manage events.

    Question-4: What do you understand by event bubbling in DOM?

    Event Bubbling => When an event occurs on an element, it first runs its handlers, then, if not stopped, it runs the handlers on that element's parents.
